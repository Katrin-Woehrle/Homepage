⇒ Aktuell in Bearbeitung: Lernprogramm Basispass Pferdekunde

Hinweise zur Installation:

Der Link führt zu meinem Dropbox-Account, auf dem sich das Lernprogramm befindet.

Ein eigener Dropbox-Account wird jedoch nicht benötigt!

Nun kann das komplette Lernprogramm als .zip runtergeladen werden, indem rechts oben auf Herunterladen geklickt wird.

Dabei kann entschieden werden, ob man es in seiner eigenen Dropbox (falls vorhanden) speichern will oder direkt auf dem PC.

Nun muss (leider noch) noch ein Ordner namens "Temp" unter C: angelegt werden, in den man die Datenbank-Datei FragenBachelor.mdb speichern muss.

Dabei muss beachtet werden, dass "Temp" richtig geschrieben wird, damit das Lernprogramm auch darauf zugreifen kann!

Dann installiert man das Lernprogramm, indem man auf setup.exe (die sich in dem runtergeladenen Ordner befindet) klickt und den Intsallationsprozess durchläuft.

Sobald die Installation erfolgreich durchlaufen wurde, startet das Lernprogramm, außerdem erscheint ab sofort "LernprogrammBasispassPferdekunde" im Startmenü.

Wenn das Programm irgendwann nicht mehr benötigt wird, kann es über die Systemsteuerung oder direkt im Startmenü wieder deinstalliert werden.

Der Ordner C:\Temp (mit der darin enthaltenen Datenbank-Datei "FragenBachelor.mdb") muss jedoch (leider noch) manuell entfernt werden.
